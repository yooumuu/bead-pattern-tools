# Bead Pattern Tools

A curated guide and resource collection for bead pattern creation — covering tools, techniques, and community resources for Perler, Hama, and Artkal bead crafting.

## What's Inside

- **Recommended Pattern Generator** — [MakeBead](https://makebead.com/) for converting images to bead patterns
- **Step-by-Step Guide** — How to go from image to finished bead art
- **Pegboard Size Reference** — Bead counts and best uses for each grid size
- **Brand Comparison** — Perler vs Hama vs Artkal detailed breakdown
- **Ironing Guide** — Techniques to avoid warping and get clean fuses
- **Pro Tips** — Color management, sorting, and workflow optimizations
- **FAQ** — Common beginner questions answered
- **Community Links** — r/beadsprites, OpenGameArt, and more
